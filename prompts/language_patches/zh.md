## Language-Specific Instructions

### Output Language
All output must be in Chinese, including:
- All titles for engaging moments
- All explanations in the "why_engaging" field
- All transcript excerpts
- All other text fields in the JSON response

### Title Writing Guidelines (标题撰写指南)

**核心原则：具体、有钩子、不夸张 — 像短视频标题，但不标题党**

#### 必须遵守的规则：
1. **具体化，不要泛泛而谈**
   - 避免「讨论」「互动」「精彩时刻」等空泛词
   - 点明人物、事件、动作或关键结论

2. **突出冲突、反转或高潮**
   - 抓住最有戏的瞬间
   - 用事实本身制造好奇，而不是堆砌情绪词

3. **可用数字增强具体性**
   - 数字让内容更可信（人数、金额、时长、步骤等）
   - 感叹号最多 1 个；没有强烈情绪时不要用

4. **捕捉关键人物和关键动作**
   - 尽量包含主播名或昵称
   - 用清晰动词：揭秘、自曝、反转、翻车、破防、拆解、回应
   - 避免夸张或攻击性动词：怒斥、炸裂、碾压、暴打 等

5. **标题长度**
   - 理想：15–25 个汉字
   - 最多不超过 30 个汉字

6. **避免**
   - 不要 emoji
   - 不要「与」「及」等书面连接词硬拼
   - 不要写成干巴巴的新闻导语
   - 不要敏感、仇恨、攻击性措辞
   - 不要夸大到与内容不符

#### 标题质量自检：
- [ ] 是否包含具体人物、事件或数字？
- [ ] 是否能引起好奇，但未过度承诺？
- [ ] 是否在 25 字以内？
- [ ] 是否避免了抽象、空泛描述？
- [ ] 读起来像短视频标题，而不是标题党？
